Tutor Inteligente 

## 📒 Descrição
Um tutor inteligente personalizado é um sistema que utiliza inteligência artificial (IA) para auxiliar alunos em sua aprendizagem de forma individualizada.

## 🤖 Tecnologias Utilizadas
IA Generativa ChatGPT para roteirização e revisão;
IA Generativa ElevenLabs para clonagem e sintetização de voz;


## 🧐 Processo de Criação
ChatGPT" foi utilizado para desenvolver o roteiro e aprimorar a narrativa do podcast envolvendo o aluno durante os estudos. "ElevenLabs" ajudou a criar uma voz sintética envolvente para a narração do podcast . O episódio é uma introdução acessível e estimulante aos estudantes que buscam entender sobre o que é IA generativas


## 💭 Reflexão (Opcional)
Este projeto foi uma experiência incrível em contar histórias para os ouvintes do podcast para que aprendam mais sobre o que são as IAs generativas
